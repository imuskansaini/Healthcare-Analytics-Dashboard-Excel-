# Healthcare Analytics Dashboard (Excel)

## Overview
This project is an interactive Healthcare Analytics Dashboard built using Microsoft Excel. It analyzes patient, hospital, and billing data through Pivot Tables, Pivot Charts, KPIs, and Slicers.

## Dashboard Features
- Total Patients
- Average Age
- Average Billing Amount
- Average Length of Stay
- Patients by Medical Condition
- Patients by Gender
- Patients by Blood Type
- Patients by Admission Type
- Patients by Insurance Provider
- Test Results Distribution
- Average Age by Medical Condition

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Development
- KPI Reporting
